kobo-registry

A package registry for kpm, the Kobo e-reader package manager.

Repository registry.toml Registry format

The registry ships definitions only — forge coordinates, release-asset names, and curated uninstall recipes. The actual KoboRoot.tgz binaries are downloaded from each package's own release page over TLS.

Use

kpm registry add github.com/wolffshots/kobo-registry
kpm registry refresh
kpm search
kpm install nickelmenu

Trusting a registry is the same trust decision as running kpm add on each of its entries yourself — a definition chooses the source repo and the uninstall recipe. See the kpm README for details.
